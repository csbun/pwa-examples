Cache all files on install

If all the files are successfully cached, then the service worker will be installed. If any of the files fail to download, then the install step will fail. This allows you to rely on having all the assets that you defined.

Cache and return requests

Fetch '/service-workers-cache/css/style.css', '/service-workers-cache/js/index.js' from CACHE and '/service-workers-cache/css/color.css' from REMOTE.
